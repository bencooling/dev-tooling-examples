# webpack: hmr

Hot module replacement is supported out of the box by the

hmr (cli & nodejs) with:  
 css and js (with side effects)
  ✅ iframe
  ✅ inline


## commands

**bundle modules**  
`webpack`

**cli**  
`npm run serve`

**nodejs api**  
`node run start`
